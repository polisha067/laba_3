3. сортировки 

- Bubble Sort - пузырьковая
проходится по массиву -> сравнивает элементы рядом -> меняет местами при правильных условиях -> больший элемент в конце

- Quick Sort - быстрая
выбираем pivot -> делим массив на 3 части -> сортируем леа и прав части(рекурс)

- Counting Sort - подсчетом
ищем мин и макс значения -> создаем счетчик -> считаем кол-во встреч числа -> возвращаем отсорт массив

- Heap Sort - пирамидальная

- Radix Sort - поразрядная

- Bucket Sort - блочная
создаем пустые блоки -> закидываем элементы по блокам -> сортируем блоки -> совмещаем блоки


4. структуры
- Stack - стек
   - push() - добавть элемент
   - pop() - удалить элемент
   - peek() - посмотреть верхний
   - is_empty() - пустота?
   - size() - размер стека

- Queue - очередь
   - enqueue() - добавть элемент
   - dequeue() - удалить элемент
   - front() - просмотр первого элемента
   - is_empty() - пустота?
   - size() - размер очереди
